You can install **AR.js** with **npm** and use in any compatible project that support npm modules (React.js, Vue.js, Next.js or whatelse), to install it run:

```
// Install with npm
npm install @ar-js-org/ar.js
```
```
// Install with yarn
yarn add @ar-js-org/ar.js
```
